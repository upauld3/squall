squall
======

a distributed computation framework written in Java and using Gradle
  similar to and pulling ideas and source from [Storm] (https://storm.incubator.apache.org/)

## Goals:
* Easy to Use
* State of Art
* Easy Web Installation
* Pure Java code base
* Minimal Dependencies
